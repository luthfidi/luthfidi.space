<h1>luthfidi.space</h1>

<p>Personal portfolio built with Next.js, TypeScript, Tailwind CSS, and next-intl.</p>

## Getting Started

```bash
npm install
cp .env.example .env.local
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
